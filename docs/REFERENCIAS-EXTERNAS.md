## Hermes Agent / Skills

- [Hermes Agent Releases](https://github.com/NousResearch/hermes-agent/releases)
  - histórico oficial de versões do runtime atual de agentes
- [Awesome OpenClaw Agents](https://github.com/mergisi/awesome-openclaw-agents)
  - biblioteca de padrões úteis para agentes, skills, SOULs, playbooks e contratos de delegação
  - manter como referência conceitual, mesmo após a migração para Hermes Agent
- [CrewClaw Multi-Agent Setup Guide](https://www.crewclaw.com/blog/openclaw-multi-agent-setup-guide)
  - referência prática para squad pequeno, workspace compartilhado, handoffs e heartbeats
- [Anthropic Skills](https://github.com/anthropics/skills/tree/main/skills)
  - referência de estrutura para skills reutilizáveis
- [Skills.sh](https://skills.sh/)
  - catálogo e descoberta de skills
- [ClawHub Skills](https://clawhub.ai/skills?sort=downloads)
  - skills populares no ecossistema legado
- [MCP Market Skills](https://mcpmarket.com/tools/skills)
  - mapeamento de skills e ferramentas que podem virar automações ou contratos mais fortes
- [Skill Finder & Installer](https://mcpmarket.com/tools/skills/skill-finder-installer)
  - skill meta para descobrir, avaliar e instalar outras skills
- [MCP Market Leaderboard](https://mcpmarket.com/tools/skills/leaderboard)
  - fonte viva para encontrar skills em alta
- [Google Workspace CLI Assistant](https://mcpmarket.com/tools/skills/google-workspace-cli-assistant)
  - referência para ampliar acesso operacional ao Workspace via CLI e automações assistidas
- [Spec Kit](https://github.com/github/spec-kit)
  - referência para constituição, specs e planos em mudanças maiores do projeto
